# Folder Logo / Gambar Aplikasi

Taruh file logo aplikasi kamu di folder ini dengan nama:
**`logo.png`**

(Pastikan namanya huruf kecil semua: `logo.png`, disarankan format PNG dengan background transparan).

Setelah menaruh file di sini, pastikan untuk **restart penuh aplikasi** (stop terminal dan ketik ulang `flutter run`) agar gambar berhasil dimuat oleh sistem Flutter.
